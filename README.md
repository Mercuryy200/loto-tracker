# Loto Tracker - Application Nuxt.js

Application web de suivi de résultats de loterie développée avec Nuxt.js 3, TypeScript et Tailwind CSS.

## Fonctionnalités

- Consultation des derniers résultats de tirages
- Analyse statistique des numéros les plus fréquents
- Générateur de numéros aléatoires
- Interface responsive et moderne
- Performance optimisée avec SSR/SSG

## Technologies

- **Nuxt.js 3** - Framework Vue.js full-stack
- **TypeScript** - Typage statique
- **Tailwind CSS** - Framework CSS utility-first
- **Composition API** - Architecture moderne Vue 3

## Installation

```bash
# Cloner le repo
git clone https://github.com/Mercuryy200/loto-tracker.git
cd loto-tracker

# Installer les dépendances
npm install

# Lancer en mode développement
npm run dev

# Build pour production
npm run build

# Générer les pages statiques
npm run generate
```
